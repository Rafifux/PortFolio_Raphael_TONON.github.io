# Projets Stage 1 - Captusite (Version statique)

Cette version est une conversion statique du site `synelva.captusandbox.com` de la section STAGE 1 (CAPTUSITE).

## Contenu

- `index.html` : page principale statique de suivi de consommation.
- `css/styles.css` : styles visuels.
- `scripts/app.js` : logique de génération du tableau et des graphiques.

## Fonctionnalités

- Tableau des consommations sur une période simulée.
- Graphique journalier avec heures creuses, heures pleines et température apparente.
- Graphique mensuel totalisé.
- Graphique des prix simulés pour HC et HP.
- Filtre de dates en JavaScript.

> Note : cette version ne se connecte pas à une base de données. Elle utilise des données d'exemple pour présenter le rendu final du site.
