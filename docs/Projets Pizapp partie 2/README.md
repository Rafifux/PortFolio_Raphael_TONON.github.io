# PizApp Partie 2 - Version statique

Cette page est une conversion statique du client Java Swing `order-status-client` de `PizApp Lot 9`.

- Interface inspirée du design Java original.
- Données locales dans `scripts/app.js`.
- Actions `Accepter`, `Refuser`, `Marquer prête` et `Rafraîchir` sont simulées côté client.

## Fichiers

- `index.html` : page principale statique.
- `css/styles.css` : styles visuels.
- `scripts/app.js` : logique d'affichage et d'interaction.

> Note : cette version ne se connecte pas à une base de données. Elle sert de démonstration statique et de présentation dans le portfolio.
